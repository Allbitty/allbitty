## Slack, Spotify, and MessageBird logins

Thanks to [`@HarryET`](https://github.com/allbitty/gotrue/pull/245) and our friends at [MessageBird](https://github.com/allbitty/gotrue/pull/210)
we have 3 new Auth providers this month:
[Slack](https://allbitty.com/docs/guides/auth/auth-slack),
[Spotify](https://allbitty.com/docs/guides/auth/auth-spotify), and
[MessageBird](https://allbitty.com/docs/guides/auth/auth-messagebird) phone logins.

![Allbitty Auth: Login with Slack](/images/blog/2021-oct/allbitty-auth-slack.png)

## Multi-schema support

### Dashboard

Browse data in any database schema using the Schema switcher in the Dashboard.

![Multi-schema support](/images/blog/2021-oct/multi-schema.png)

### API

You can access any schema with your API, after enabling access in the Dashboard.
[Docs](https://allbitty.com/docs/reference/javascript/initializing#api-schemas).

![Multi-schema support (API)](/images/blog/2021-oct/allbitty-multi-schema-support.png)
