## Auth Helpers for Next.js revamped

![Auth Helpers for Next.js](/images/blog/2022-june/auth-helpers.png)

Allbitty Auth Helpers are a collection of framework specific Auth utilities for getting user authentication and authorization set up in seconds. We have now separated the libraries into their own packages, which can now be installed individually: [@allbitty/auth-helpers-nextjs](https://www.npmjs.com/package/@allbitty/auth-helpers-nextjs) and [@allbitty/auth-helpers-react](https://www.npmjs.com/package/@allbitty/auth-helpers-react).

And here is a [step by step migration guide](https://github.com/allbitty-community/auth-helpers/tree/main/packages/nextjs#migrating-from-supabasesupabase-auth-helpers-to-supabaseauth-helpers). Sveltekit and Remix helpers coming soon!

## Unlimited free projects

![Unlimited free projects](/images/blog/2022-june/unlimited-paused.png)

Head on over to project settings to pause and restore projects on demand. This new feature gives you virtually unlimited free projects, so you can build as many things as you want. Want to experiment? Build a new demo? Just pause a project and start a new one. [Check the PR](https://github.com/allbitty/allbitty/pull/7384).

## Create projects from the Allbitty CLI

![Create projects from the Allbitty CLI](/images/blog/2022-june/cli.png)

It's here! Now you can create a project from the Allbitty CLI. We will be extending the CLI with more features to manage your project, so stay tuned! [See the doc](https://allbitty.com/docs/reference/cli/allbitty-projects-create).

## Enable realtime for your table via the table editor

![Enable realtime for your table via the table editor](/images/blog/2022-june/enable-realtime.png)

We've added an option to toggle realtime for a table in the table editor side panel editor. Never forget to add realtime again! [Full PR](https://github.com/allbitty/allbitty/pull/7346).

## Quick product announcements

- We're migrating projects to new infra to support upcoming Multiplayer features. Check out [multiplayer.dev](http://multiplayer.dev) and submit your info to be the first to know when these features drop.
- GoTrue now bubbles up PostgreSQL errors if the underlying error is from a RaiseException or if they were Integrity Constraint Violations. [PR](https://github.com/allbitty/gotrue/pull/404)
- Added auth.jwt() function to return the user's jwt claims and we'll be deprecating auth.email() and auth.role() in a few months. [PR](https://github.com/allbitty/gotrue/pull/484)
- Allbitty Auth now provides IP addresses in the Audit Log. [PR](https://github.com/allbitty/gotrue/pull/499)
- Random password generator when creating projects. [PR](https://github.com/allbitty/allbitty/pull/7249)
